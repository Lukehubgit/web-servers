About This Project

This site was built for IT 3203 Web Development at KSU. The assignment had three milestones, each building on the last. Everything was written by hand with no templates or frameworks. It is deployed and hosted on GitHub Pages.

Milestone 1 - Foundation

Built the core HTML structure and CSS stylesheet from scratch. Set up the page layout, navigation, color scheme, and basic content across all seven pages. Established the overall look and feel of the site.

Milestone 2 - Interactivity

Added a JavaScript-powered self-assessment quiz with pass/fail logic and a score display. Expanded research content across all pages, added tables, callout boxes, stat cards, and a page hero section to make the content more organized and readable.

Milestone 3 - Responsive Design

Made the entire site mobile-friendly using CSS media queries for two screen sizes: tablet (768px) and phone (480px). Added a hamburger menu for mobile navigation, moved JS into an external file, and applied responsive best practices to the layout, images, tables, and quiz form.

Tech Stack This site uses plain HTML, CSS, and JavaScript with no libraries or frameworks. The quiz logic, hamburger menu, and all styling were written from scratch. Source code is available on GitHub and the site is hosted for free using GitHub Pages.

This is a class project for IT 3203 Web Development at Kennesaw State University.
