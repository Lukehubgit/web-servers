About This Project

This site was built for IT 3203 Web Development at KSU. The project has three milestones. Milestone 1 was the base HTML and CSS site. Milestone 2, which is what you are looking at now, added a JavaScript-powered self-assessment quiz and expanded the content across all the pages. Milestone 3 will focus on making the site fully responsive on mobile.

Everything was written by hand — no templates, no frameworks. It is hosted on GitHub Pages.

This is a class project for IT 3203 Web Development at Kennesaw State University.
