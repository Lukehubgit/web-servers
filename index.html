Welcome

This website is about web servers, which are one of the most important technologies behind the modern Internet. A web server stores website files and sends them to users through their browsers using HTTP. If web servers did not exist, there would be no way to access websites or really anything online.

This site goes over how web servers work, the history of how they got to where they are today, some key concepts to know, and real examples like Apache, Nginx, and IIS.

Key Terms

HTTP

HyperText Transfer Protocol, which is the communication language used between browsers and servers.

Static Server

Delivers files exactly as they are stored, with no extra processing involved.

Dynamic Server

Generates content in real time using application servers and databases.

URL

Uniform Resource Locator, basically the address used to find a resource on a server.

Apache / Nginx

Two of the most popular web server software options used across the internet today.

Client-Server Model

The basic setup where a browser (client) requests content and a server sends it back.
